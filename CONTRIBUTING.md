# Bijdragen aan onze website OpenNederland.nl

Onze website is uiteraard ook open source. Iedereen kan verbeteringen voorstellen, typo's corrigeren, initiatieven toevoegen of zijn eigen ledenprofiel bijwerken. Voor ontwikkelaars: nieuwe functionaliteit is ook welkom, maak daarvoor eerst wel een [issue](https://github.com/opennederland/www.opennederland.nl/issues) aan waarin je beschrijft wat je wilt doen.

Let wel, al je bijdragen in code zijn EUPL V1.2, alle bijdragen in tekst (content) zijn CC BY-SA 4.0

### 1. Een bestaande pagina bewerken
Zie je een foutje of wil je je profiel updaten?
1. Ga naar de pagina op [OpenNederland.nl](https://www.opennederland.nl).
2. Scroll naar beneden (bij ledenprofielen en organisatieprofielen) en klik op de knop **"Bewerk dit profiel op GitHub"** (of zoek het bestand handmatig in deze repository).
3. Klik op het potlood-icoontje âœï¸ rechtsboven in het bestand.
4. Voer je wijzigingen door.
5. Klik op **"Commit changes..."**.
6. Selecteer **"Create a new branch for this commit and start a pull request"** en klik op **Propose changes**.

Een beheerder kijkt je wijziging na en zet het live!

### Ledenprofielen (`_members`)
Ben je lid en wil je je profiel aanpassen?
* Je vindt de profielen in de map [`_members/personen`](./_members/personen) of [`_members/organisaties`](./_members/organisaties).
* Bewerk je bestaande bestand. Lid worden? Vul dan het formulier in op onze [website](https://www.opennederland.nl/vereniging/#lid-worden).
* Elk bestand begint met een stukje metadata (Front Matter) tussen `---` streepjes. 
* Alles wat onder de `---` streepjes staat wordt getoond, schrijf dit in het markdown formaat. Weet je niet hoe je markdown schrijft? Je kan deze [open source editor](https://stackedit.io/) gebruiken.
